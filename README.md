# Cinema Colombia Frontend

Este es el proyecto frontend para la plataforma de compra de entradas de cine.

## Tecnologías
- **React 18**
- **Vite**
- **TypeScript**
- **Lucide Icons**
- **Vanilla CSS** (Custom Design System)

## Instalación
```bash
npm install
```

## Ejecución
```bash
npm run dev
```
